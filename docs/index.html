📖 Sobre o Projeto

File to Video Encoder é uma ferramenta inovadora que permite codificar qualquer arquivo binário em um vídeo visual, e posteriormente decodificá-lo de volta ao arquivo original com verificação de integridade.

🎯 Características Principais

🔌
Arquitetura de Plugins

Sistema dinâmico que descobre automaticamente novos encoders

🔐
Assinatura Digital

Cada encoder tem assinatura única nos primeiros pixels

📊
Métricas em Tempo Real

Tamanho, taxa de compressão e tempo de processamento

📜
Histórico de Ações

Registro de todas as codificações e decodificações

📤 Como Usar - Encoder

  1. Acesse a aba Encoder
  2. Faça upload do arquivo que deseja codificar
  3. Escolha o método de codificação na barra lateral
  4. Clique em "Iniciar Codificação"
  5. Acompanhe as métricas em tempo real
  6. Baixe o vídeo gerado

📥 Como Usar - Decoder

  1. Acesse a aba Decoder
  2. Faça upload do vídeo codificado ou use o último vídeo gerado
  3. O sistema identifica automaticamente o formato pela assinatura
  4. Clique em "Decodificar"
  5. Baixe o arquivo recuperado

🔌 Plugins Disponíveis

🎨
RGB Tabuleiro
Padrão de tabuleiro colorido. Cada célula armazena 3 bytes (RGB). Visual artístico.
📦 AVI 🎨 3 bytes/pixel RGB1
📊
YUV Barras
Barras de luminância em escala de cinza. Mais rápido e compacto.
📦 AVI ⚡ 1 byte/pixel YUV1
📱
QR Code Sequence
Sequência de QR Codes. Resistente à compressão do YouTube.
📦 AVI 🔐 Reed-Solomon QRC1
⚡
RGB FAST
Compressão zlib + redundância. YouTube Proof e ultra rápido.
📦 MP4 🗜️ zlib FST1
💡 Sistema de Plugins Dinâmico: Para adicionar um novo encoder, basta criar uma pasta em encoders/ com encoder.py contendo uma classe que herda de BaseEncoder. O sistema descobre automaticamente!

🔐 Sistema de Identificação por Assinatura

Cada encoder possui uma assinatura única de 4 bytes que é embutida nos primeiros pixels (canto superior esquerdo, 2x2 pixels) de cada frame do vídeo. Isso permite a identificação automática do formato durante a decodificação.

🔍 Como Funciona

  1. O encoder define uma assinatura única (ex: b'RGB1')
  2. Durante a codificação, a assinatura é embutida nos pixels 2x2 do canto superior esquerdo
  3. Na decodificação, o sistema extrai esses pixels e compara com as assinaturas registradas
  4. O encoder correspondente é automaticamente selecionado para decodificação

📋 Assinaturas Registradas

Encoder Assinatura Bytes (Hex) Visual (2x2 pixels)
🎨 RGB Tabuleiro b'RGB1' 52 47 42 31
📊 YUV Barras b'YUV1' 59 55 56 31
📱 QR Code Sequence b'QRC1' 51 52 43 31
⚡ RGB FAST b'FST1' 46 53 54 31

💻 Desenvolvimento de Plugins

📝 Criando um Novo Encoder

Para adicionar um novo método de codificação, siga estes passos:

  1. Crie uma pasta em encoders/meu_encoder/
  2. Crie encoder.py com uma classe herdando BaseEncoder
  3. Crie __init__.py exportando a classe
  4. (Opcional) Crie decoder.py com a função de decode

Exemplo de Encoder

import numpy as np

class MeuEncoder(BaseEncoder):
    
    # Propriedades obrigatórias
    @property

        return "meu_encoder"
    
    @property

        return "🎯 Meu Encoder"
    
    @property
    def description(self) -> str:

    @property
    def icon(self) -> str:
        return "🎯"
    
    @property
    def output_extension(self) -> str:
        return "avi"
    
    @property
    def mime_type(self) -> str:
        return "video/avi"
    
    @property
    def signature(self) -> bytes:
        return b'MEU1'  # Assinatura única de 4 bytes
    
    def encode(self, data, file_path, progress_callback):
        # Implemente a codificação
        pass
    
    def decode(self, video_data):
        # Implemente a decodificação
        pass

📚 API Reference

🔌 BaseEncoder

Propriedade/Método Tipo Descrição
name property Identificador único do encoder
display_name property Nome exibido na interface
signature property Assinatura única de 4 bytes
encode() method Codifica dados em vídeo
decode() method Decodifica vídeo para dados
can_decode() method Verifica assinatura no vídeo
embed_signature() method Embute assinatura nos pixels

📦 EncoderRegistry

Método Descrição
register(encoder_class) Registra um encoder no sistema
identify(video_data) Identifica formato pela assinatura
list_registered() Lista todos os encoders registrados

📊 Sistema de Métricas e Histórico

📈 Métricas em Tempo Real

  • Tamanho Original: Tamanho do arquivo antes da codificação
  • Tamanho do Vídeo: Tamanho do vídeo gerado
  • Taxa de Expansão: Relação entre tamanho do vídeo e original
  • Tempo de Processamento: Duração da codificação/decodificação

📜 Histórico de Ações

O sistema mantém um histórico das últimas 20 ações, incluindo:

  • Tipo de ação (encode/decode)
  • Método utilizado
  • Nome do arquivo
  • Tamanhos antes e depois
  • Tempo de processamento
  • Timestamp
